## ‚òÅÔ∏è **Frontend Secrets (Cloudflare)**

### **9. CLOUDFLARE_API_TOKEN**

**How to get:**
1. Go to: https://dash.cloudflare.com/profile/api-tokens
2. Click **"Create Token"**
3. Use **"Edit Cloudflare Workers"** template
4. Add permission: **Account** ‚Üí **Cloudflare Pages** ‚Üí **Edit**
5. Click **"Continue to summary"** ‚Üí **"Create Token"**
6. Copy the token and paste here

### **10. CLOUDFLARE_ACCOUNT_ID**

**How to get:**
1. Go to: https://dash.cloudflare.com/
2. Click on your domain
3. Scroll to **"API"** section
4. Copy **"Account ID"**
5. Paste here
